# utez-2e-biblioteca-javafx-equipo01
Biblioteca Tarea Integradora 

Este proyecto es sobre el desarrollo de un sistema de Biblioteca Escolar para ayudar a la gestión del catálogo de los libros donde puedes hacer operaciones básicas como resgistrar el libro, cosultar, editar y eliminar los libros. Asi como ver si se encuentran disponibles.

Además incluye validaciones de datos para que fluya correctamente el registro y los datos se guarden orrectamente sin que hayan bugs.
